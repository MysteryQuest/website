OSINT Toolkit

Open Source Intelligence for Evidence-Based Investigation

What is OSINT?

Open Source Intelligence (OSINT) refers to the collection and analysis of information from publicly available sources. At The Unverified File, we use OSINT techniques to verify claims, investigate mysterious phenomena, and separate fact from fiction.

Our methodology combines automated AI analysis with human verification, ensuring rigorous standards for evidence evaluation.

Geospatial Intelligence

  • Google Earth Pro: Historical satellite imagery analysis
  • Sentinel Hub: Real-time satellite data (ESA Copernicus)
  • OpenStreetMap: Crowdsourced geographic data
  • SunCalc: Sun position, shadows, and timing verification
  • MarineTraffic: Vessel tracking and maritime data

Image & Video Analysis

  • InVID: Reverse video search and forensic analysis
  • FotoForensics: Error Level Analysis (ELA) for image manipulation
  • TinEye: Reverse image search
  • Jeffrey's Exif Viewer: Metadata extraction
  • FFmpeg: Video frame extraction and analysis

Social Media Intelligence

  • Social Searcher: Real-time social media monitoring
  • Wayback Machine: Historical website archives
  • Reddit Search: Advanced Reddit post analysis
  • Twitter Advanced Search: Timeline and keyword filtering
  • YouTube DataViewer: Video upload verification

Data Analysis & AI

  • OpenAI GPT: Text analysis and pattern recognition
  • Claude (Anthropic): Document analysis and summarization
  • Python OSINT Libraries: Automated data collection
  • Jupyter Notebooks: Reproducible analysis workflows
  • Maltego: Link analysis and data visualization

Our Investigation Process

1

Initial Assessment

Evaluate source credibility, gather initial evidence, and establish investigation parameters.

2

Data Collection

Use OSINT tools to gather publicly available information from multiple independent sources.

3

AI-Powered Analysis

Apply machine learning and pattern recognition to identify anomalies, correlations, and inconsistencies.

4

Human Verification

Expert review and community validation to ensure accuracy and prevent AI hallucination.

5

Report & Publication

Publish findings with full transparency, including source links and confidence levels.

Ethical Guidelines

  • We only use publicly available information
  • We respect privacy and do not dox individuals
  • We cite all sources and provide verification links
  • We acknowledge uncertainty and avoid speculation
  • We correct errors promptly and transparently
